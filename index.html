Barcharts (U.S. Economic Data)

To illustrate a realistic use-case for Redflare bar charts we used publically available data to create a few visulizations.

These time series bar charts can be customized to show more or fewer labels (on the x axis) based on the size of the chart. The Redflare code will sort the data by date and provide the option to show more or fewer date points along the x axis. Redflare also provides custom tooltip formatting.

The US Census Bureau conducts an anual survey which is named the CPS (Current Population Survey). Over the years the U.S. Population has grown, and with it the number of households surved has increased as shown in the following chart.
